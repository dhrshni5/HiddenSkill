# 🧠 HiddenSkill

**HiddenSkill** is an AI-inspired career analysis system that helps users discover their hidden strengths and identify suitable career paths based on behavioral patterns.

---

## 🚀 Features

* 🧠 Personality analysis based on user responses
* 💪 Strengths and ⚠️ weaknesses identification
* 🚀 Career recommendations with explanations
* 🎯 Career fit score visualization
* 📄 Downloadable career report
* 🎨 Clean and modern UI

---

## 🧠 Concept

This project uses a **rule-based AI approach (behavioral inference system)** to analyze user inputs and generate personalized career insights.

Instead of relying on heavy machine learning models, the system focuses on:

* Pattern recognition
* Decision logic
* Interpretable AI reasoning

---
